# Installation Standards

## Plot-first assumptions

Vacant land requires outdoor resilience. Installations should assume inconsistent power, harsh weather, limited Wi-Fi, and a need for elevated visibility.

## Baseline recommendations

- Prefer solar-powered LTE cameras for independent operation
- Mount primary cameras high enough to resist easy tampering
- Cover gates, likely approach paths, and boundary weak points first
- Record exact install location in the device record
- Use labels and serial capture during onboarding

## Placement guidance

- Walled plot: cover gate and at least one opposite-angle boundary view
- Partially fenced plot: prioritize exposed edges and access gaps
- Open plot: use pole-mounted visibility and reduce blind approaches

## Maintenance requirements

- Validate heartbeat after installation
- Capture battery baseline
- Confirm night visibility and rain tolerance
- Review site health regularly for silent failures
