# Escalation Matrix

## Severity mapping

### Info
- Store for timeline
- No dispatch
- Owner notification only if policy requires

### Warning
- Review in ops queue
- Investigate device health or pattern buildup
- Notify owner according to escalation plan

### Critical
- Create or elevate incident immediately
- Queue owner notifications
- Verify rapidly
- Dispatch partner when field confirmation is required

## Stakeholder communication

- Owner: must know what happened, when, and what action is underway
- Delegate: inform when the owner has chosen delegate involvement
- Partner: receive only the dispatch-relevant context
- Internal ops: record every major state transition
