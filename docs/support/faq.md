# FAQ

## Is this a smart home system?
No. The MVP is designed for vacant land plots, not occupied homes.

## Does the system send real WhatsApp or SMS messages today?
The current implementation logs notification intent and is ready for provider integration later.

## Can delegates manage properties?
Delegates can be invited and assigned to properties, but owner and ops roles remain the main authority roles in the current implementation.

## Does the system store actual video files?
The MVP stores evidence metadata and storage references. A production media pipeline is a future step.

## Can I use this with multiple plots?
Yes. The data model supports multiple properties per owner.
