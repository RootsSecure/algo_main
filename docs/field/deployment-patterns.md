# Plot Deployment Patterns

## Pattern 1: Walled plot with gate

Best for:
- clear entry point detection
- gate tamper correlation
- lower ambiguity around legal access points

Suggested setup:
- gate-facing camera
- boundary-facing camera if budget allows
- gate sensor or tamper sensor
- optional siren or smart lock if physical gate supports it

## Pattern 2: Partially fenced plot

Best for:
- mixed access-risk environments
- growth-perimeter areas under development

Suggested setup:
- elevated camera on the most exposed edge
- one additional view on likely vehicle or footpath approach
- stronger ops review rules for ambiguous motion

## Pattern 3: Open plot

Best for:
- early land holding without boundary works

Suggested setup:
- high-mounted outdoor camera
- explicit blind-spot documentation
- heavier reliance on repeated-event patterns and human verification
