# Documentation Map

This directory contains the beginner-friendly documentation baseline for the NRI Plot Sentinel backend.

## Start here

- `../README.md`: setup, commands, and project overview
- `getting-started.md`: plain-English first tour of the project
- `../CONTRIBUTING.md`: how to make safe changes

## Documentation sections

- `product/`: vision, problem statement, PRD, personas, journeys, roadmap
- `architecture/`: system overview, domain model, event flow, security model, ADRs
- `api/`: endpoint catalog, auth rules, error model, webhook and gateway contracts
- `ops/`: triage, dispatch, and escalation workflows
- `field/`: installation and deployment guidance for vacant plots
- `qa/`: test strategy, acceptance cases, and manual validation
- `support/`: onboarding, glossary, FAQ, troubleshooting, and app prompts

## Best reading order for beginners

1. `getting-started.md`
2. `product/problem-statement.md`
3. `architecture/system-overview.md`
4. `api/endpoint-catalog.md`
5. `api/raspberry-pi-gateway.md`
