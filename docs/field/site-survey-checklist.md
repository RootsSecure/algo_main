# Site Survey Checklist

## Before install

- Confirm exact plot boundaries
- Confirm address and map pin
- Identify likely intrusion paths
- Check mobile network strength
- Confirm sun exposure for solar devices
- Identify safe mounting points
- Note nearby obstructions and blind spots

## During install

- Record serial numbers
- Record mounting heights and directions
- Validate heartbeat and network status
- Validate media capture
- Capture baseline site photos

## After install

- Verify property and device records are linked correctly
- Confirm escalation settings with the owner
- Confirm delegate contacts if applicable
- Confirm health endpoints reflect the new device state
