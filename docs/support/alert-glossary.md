# Alert Glossary

- `motion`: movement detected near the property or zone
- `tamper`: attempted interference with device or mount
- `offline`: device lost coverage or stopped checking in
- `gate_breach`: gate or access-point breach event
- `manual_report`: incident entered by a human source

## Severity meanings

- `info`: timeline event with low operational urgency
- `warning`: notable event or health issue that needs review
- `critical`: event that likely requires immediate owner attention and possible escalation

## Verification meanings

- `pending`: not yet confirmed
- `auto_verified`: system rules marked it strong enough to elevate
- `ops_verified`: human operator confirmed it
- `dismissed`: reviewed and not considered an active threat
